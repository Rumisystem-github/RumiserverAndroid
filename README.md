# 仕組み
AIDLを使用して、ユーザー情報や通知を取得します。<BR>
AIDLに接続できるアプリは制限されます<BR>
![](Screenshot_20250615_163310.png)
ForegoundServiceはるみ鯖APIとの仲介役です。<BR>
ちなみにコイツがWebSocketにも常時接続しています。<BR>
